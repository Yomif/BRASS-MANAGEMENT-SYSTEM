# BRASS-MANAGEMENT-SYSTEM
 🛒 Brass Store Shopping System

A modern desktop Point of Sale (POS) application developed with Python, CustomTkinter, SQLite, and PIL.

The system allows store owners to manage shopping carts, generate receipts, track sales records, and improve customer checkout experience through a clean graphical interface.


 ✨ Features

- 🔐 Secure Login System
- 🛍️ Add Items to Shopping Cart
- 📦 Product Category Selection
- 💰 Automatic Discount Calculation
- 🧾 Professional Receipt Generation
- 💾 Save Receipt as Text File
- 🌗 Dark/Light Mode Toggle
- 🗄️ SQLite Database Integration
- 🎨 Modern CustomTkinter UI
- 📊 Sales Record Storage

🖥️ Technologies Used

- Python
- CustomTkinter
- SQLite3
- Pillow (PIL)
- Tkinter
